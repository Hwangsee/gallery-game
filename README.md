## 📚 기술 스택

- React 19
- TypeScript
- Vite
- React Router v7
- Ant Design (antd)
- Axios

---

## 🛠 설치 및 실행 방법

```
yarn install
yarn dev
```

---

## 📆 사용한 라이브러리 및 사용 이유

| 라이브러리        | 사용 이유                                |
| ----------------- | ---------------------------------------- |
| Ant Design (antd) | 빠른 UI 컨퍼런트 구축 및 일관된 스타일링 |
| React Router Dom  | 페이지 이동 및 라우팅 처리               |
| Axios             | API 호출 시 안정적                       |
| Vite              | 빠른 개발 서버 구동을 위해 사용          |
